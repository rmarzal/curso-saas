# Briefs visuales · 10 piezas

Para Canva, Figma o un diseñador externo. Cada brief incluye paleta NS, tipografía,
mensaje principal, formato y uso recomendado. Usa los tokens del archivo
`servidor/metricas/_design.css` como referencia visual.

## Paleta a usar (NextScenario)

```
Background:  #F6F4EE  (cream paper)
Background2: #EBE7DB  (cream darker)
Ink:         #0A1230  (deep navy · texto principal)
Lime accent: #D4FF3F  (highlights, pills, dot pulsante)
Blue accent: #5B8CFF  (electric blue · datos secundarios)
Mute:        rgba(10,18,48,0.58) (texto secundario)
```

## Tipografía

- **Geist** — sans-serif para titulares (weight 500-600).
- **Instrument Serif italic** — para acentos y citas.
- **JetBrains Mono** — para etiquetas, números, eyebrows.

Las 3 están en Google Fonts gratis.

**Paleta exacta (copy-paste para Canva/Figma):**
```
Cream paper:   F6F4EE
Cream paper-2: EBE7DB
Ink navy:      0A1230
Ink-2:         111B42
Lime accent:   D4FF3F
Blue accent:   5B8CFF
```

**Logo NS para reusar:**
- Dos cuadrados de 22×22 px, esquinas radius 4px.
- Cuadrado 1: ink (`#0A1230`), translate(-3px, -3px).
- Cuadrado 2: lime (`#D4FF3F`), translate(3px, 3px), mix-blend-mode: multiply.

Si lo haces en Canva: dibuja los 2 cuadrados y aplica el modo de mezcla "multiply"
en el segundo. El resultado son 3 colores visibles (ink, lime y un overlap oscuro).
